# 파이썬 & Git 기초 마스터 퀴즈 게임

## 1. 프로젝트 개요
이 프로젝트는 파이썬(Python)의 핵심 문법과 Git 버전 관리 시스템의 기초를 복습할 수 있는 콘솔 기반의 인터랙티브 퀴즈 게임입니다. 단순히 코드를 따라 치는 것을 넘어, 객체 지향 프로그래밍(OOP) 패러다임을 적용하여 클래스 단위로 역할을 분리하고, JSON 파일 입출력을 통해 데이터를 영구적으로 저장하는 '데이터 영속성'을 직접 구현했습니다. 또한, 전체 개발 과정을 Git으로 관리하며 브랜치 생성, 병합(Merge) 등의 실전 협업 워크플로우를 경험하는 것을 목표로 제작되었습니다.

## 2. 퀴즈 주제 선정 이유
본격적인 소프트웨어 개발을 심도 있게 학습하는 3학년 과정에 접어들면서, 새로운 기술을 배우는 것만큼이나 개발의 가장 기본이 되는 핵심 도구들을 완벽하게 다루는 것이 중요하다는 것을 체감했습니다. 파이썬의 자료형, 제어문, 그리고 Git의 원격 저장소 관리 명령어 등은 매일 사용하는 필수 지식이므로, 이를 퀴즈 형태로 스스로 묻고 답하며 확실하게 내재화하기 위해 이 주제를 선정했습니다. 내가 직접 구조를 짠 프로그램으로 나의 성장을 돕는 도구를 만들었다는 점에서 큰 의미가 있습니다.

## 3. 주요 기능 목록

### 🎯 기본 기능
* **인터랙티브 메뉴 인터페이스**: 직관적인 번호 선택을 통해 프로그램의 모든 기능(풀기, 추가, 목록, 확인, 종료)에 쉽게 접근할 수 있습니다.
* **퀴즈 풀기 및 채점**: 등록된 퀴즈를 풀고, 입력한 답과 정답을 대조하여 즉시 채점 결과를 제공합니다.
* **새로운 퀴즈 동적 추가**: 프로그램 실행 중에도 사용자 입력을 받아 새로운 문제, 4개의 선택지, 정답 및 힌트를 자유롭게 추가할 수 있습니다.
* **전체 퀴즈 목록 조회**: 현재 시스템에 등록된 모든 퀴즈의 문제와 정답을 한눈에 확인할 수 있습니다.
* **점수 관리 및 데이터 영속성 (`state.json`)**: 게임 종료 후에도 새로 추가한 퀴즈와 역대 최고 점수 데이터가 `state.json` 파일에 안전하게 보존되며, 다음 실행 시 자동으로 로드됩니다.
* **강건한 예외 처리**: 사용자의 빈 입력, 문자 입력, 범위를 벗어난 숫자 등 비정상적인 입력이나 강제 종료(Ctrl+C) 시에도 프로그램이 다운되지 않고 안전하게 데이터를 저장합니다.

### 🚀 보너스 기능 (선택 구현 사항 적용 완료)
* **문제 랜덤 출제 및 개수 선택**: 매번 같은 순서로 문제를 푸는 지루함을 없애기 위해 퀴즈를 무작위로 섞어 출제하며, 사용자가 풀고 싶은 문제의 개수를 직접 지정할 수 있습니다.
* **힌트 시스템 및 페널티**: 퀴즈 풀이 중 'h'를 입력하여 힌트를 열람할 수 있으며, 힌트 사용 시 해당 문제의 획득 점수가 절반으로 감소하는 게임적 요소를 적용했습니다.
* **퀴즈 삭제 기능**: 불필요하거나 잘못 입력된 퀴즈를 번호 선택으로 간편하게 삭제하고 저장소에 즉각 반영합니다.
* **상세 플레이 히스토리 기록**: 단순한 최고 점수뿐만 아니라, 플레이 날짜, 푼 문제 수, 획득 점수 등 최근 5회의 세부 플레이 기록을 누적하여 저장하고 조회할 수 있습니다.

## 4. 실행 방법 및 요구 환경

### 요구 환경
* **Python 버전**: Python 3.10 이상 권장
* **의존성**: 표준 라이브러리(`json`, `random`, `datetime`, `sys`)만을 사용하므로 별도의 외부 라이브러리 설치가 필요하지 않습니다.

### 실행 방법

### 가상환경(venv) 설정 (권장)
본 프로젝트는 시스템 파이썬 환경에 영향을 주지 않고 독립적인 실행 환경을 유지하기 위해 가상환경(`venv`) 사용을 권장합니다.

1. **가상환경 생성**
   ```bash
   # 프로젝트 루트 폴더에서 실행
   python -m venv venv
   ```

2. **가상환경 활성화**
   - **Windows**:
     ```bash
     .\venv\Scripts\activate
     ```
   - **macOS / Linux**:
     ```bash
     source venv/bin/activate
     ```

3. **가상환경 활성화 확인**
   - 활성화되면 터미널 프롬프트 앞에 `(venv)`라는 표시가 나타납니다.

4. **가상환경 종료**
   - 작업이 끝난 후 가상환경을 나가려면 아래 명령어를 입력합니다.
     ```bash
     deactivate
     ```

5. 저장소를 로컬 환경으로 복제(Clone)합니다.
   ```bash
   git clone [저장소 URL]
   ```
6. 프로젝트 디렉터리로 이동합니다.

7. 터미널 또는 커맨드 라인에서 아래 명령어를 입력하여 프로그램을 실행합니다.
   ```bash
   python main.py
   ```

## 6. 데이터 관리 및 파일 설명 (`state.json`)

### 파일 경로 및 역할
* **경로**: 프로젝트 루트 디렉터리 (`./state.json`)
* **역할**: 프로그램을 종료하거나 시스템을 재시작해도 데이터가 유지될 수 있도록 모든 상태를 UTF-8 인코딩의 JSON 형식으로 영구 저장합니다. 파일이 존재하지 않거나 손상된 경우 프로그램은 자동으로 기본 데이터를 생성하여 복구합니다.

### 데이터 구조 (스키마)
데이터 파일은 아래와 같은 키-값 구조로 구성됩니다.

* **`best_score`** (Number): 현재까지 달성한 최고 점수 기록입니다.
* **`history`** (Array): 최근 플레이한 게임 기록들의 리스트입니다.
    * `date`: 플레이 시각
    * `played`: 풀이한 문제 수
    * `score`: 획득 점수
* **`quizzes`** (Array): 시스템에 등록된 모든 퀴즈 객체들의 리스트입니다.
    * `question`: 퀴즈 문제 텍스트
    * `choices`: 4개의 선택지가 담긴 리스트
    * `answer`: 정답 번호 (1~4)
    * `hint`: 문제 풀이에 도움을 주는 힌트 내용

**구성 예시:**
```json
{
    "best_score": 100,
    "history": [
        {
            "date": "2026-04-22 19:30:00",
            "played": 5,
            "score": 50
        }
    ],
    "quizzes": [
        {
            "question": "Git에서 변경 사항을 원격 저장소로 올리는 명령어는?",
            "choices": ["git pull", "git push", "git commit", "git clone"],
            "answer": 2,
            "hint": "밀어넣는다(push)는 의미를 가집니다."
        }
    ]
}
```
